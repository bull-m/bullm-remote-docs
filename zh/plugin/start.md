# 插件

- 插件是我们系统的核心扩展机制，允许根据自己的需求定制功能和界面，他是直接和用户产生交互的组件
- 插件配置存储在客户端，每个小车都可独立配置，支持导入导出

## 插件的优势

✨ **模块化设计** - 每个插件独立运行，互不影响  
⚡ **即插即用** - 安装后立即生效，无需重启   
🔧 **易于开发** - 只需简单调用提供的api

## 内置插件

我们提供了一些开箱即用的内置插件：

| 名称   | 功能描述          |
|------|---------------|
| 实时视频 | 显示小车摄像头实时画面   |
| 摇杆控制 | 虚拟摇杆控制小车移动    |
| 手柄控制 | 使用游戏手柄控制小车运动  |
| 键盘控制 | 键盘按键控制小车运动    |
| 连接状态 | 显示设备连接状态和网络信息 |
| 滑块控制 | 通过滑块控制小车的设备   |
| ...  | ...           |
